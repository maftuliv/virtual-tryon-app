# Виртуальная Примерочная - IDM-VTON

Веб-приложение для виртуальной примерки одежды с использованием технологии IDM-VTON (Improving Diffusion Models for Authentic Virtual Try-on in the Wild).

## Возможности

- Загрузка 1-4 фотографий пользователя
- Загрузка изображения желаемой одежды
- Генерация фото пользователя в выбранной одежде
- Скачивание результатов
- Современный и интуитивный интерфейс

## Технологии

### Backend
- Python 3.9+
- Flask (веб-сервер)
- IDM-VTON через Gradio Client
- Pillow (обработка изображений)

### Frontend
- HTML5
- CSS3 (современный дизайн)
- Vanilla JavaScript
- Drag & Drop поддержка

## Установка и запуск

См. файл SETUP_INSTRUCTIONS.md для подробной инструкции.

## Использование

1. Откройте браузер и перейдите по адресу `http://localhost:5000` или откройте `frontend/index.html`
2. Загрузите 1-4 фотографии себя (в полный рост)
3. Загрузите фото желаемой одежды
4. Нажмите "Примерить одежду"
5. Дождитесь обработки (может занять 30-60 секунд)
6. Скачайте результаты

## API Endpoints

- `GET /` - Информация о сервере
- `GET /api/health` - Проверка состояния
- `POST /api/upload` - Загрузка изображений
- `POST /api/tryon` - Выполнение виртуальной примерки
- `GET /api/result/<filename>` - Получение результата
- `POST /api/cleanup` - Очистка старых файлов

## Известные ограничения

- Обработка может занять время в зависимости от API
- Требуется интернет-соединение для работы с моделью
- Качество результата зависит от качества входных изображений

## Лицензия
